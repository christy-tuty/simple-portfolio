# MANTA Portfolio: Data Analyst

A modern, dark-themed portfolio website designed for a Data Analyst, featuring project showcases and contact information.

## Features
- **Modern Dark Theme**: Default dark aesthetic with high-contrast accents.
- **Theme Toggle**: Switch between light and dark modes with a simple click.
- **Responsive Grid**: Clean project showcase grid layout.
- **Optimized Typography**: Built with IBM Plex Sans for professional readability.

## Technologies Used
- HTML5
- CSS3 (Variables, Grid, Flexbox)
- JavaScript

## Deployment
This project is deployed using [GitHub Pages](https://pages.github.com/).
