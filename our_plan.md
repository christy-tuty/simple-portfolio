# Project Plan: Data Analyst Portfolio Website (MANTA)

## Objective
Create a modern, dark-themed portfolio website for a data analyst to showcase skills and projects, hosted on GitHub Pages.

## Design Specifications
- **Theme:** Modern Dark Theme.
- **Typography:** IBM Plex Sans.
- **Sections:**
  1. **Hero:** "MANTA | Data Analyst"
  2. **Projects:** 3 Featured Projects (Grid layout).
  3. **Contact:** Email/LinkedIn/GitHub links.

## File Structure
- `index.html` (Main structure)
- `style.css` (Dark theme, IBM Plex Sans font import, layout)
- `assets/` (Images/Icons for projects)

## Implementation Steps

### Phase 1: Local Development
1. **HTML Layout:** Create semantic HTML structure (header, main, footer).
2. **CSS Styling:** 
   - Import IBM Plex Sans via Google Fonts.
   - Implement CSS variables for dark color palette.
   - Create responsive grid for project cards.
3. **Content Integration:** Fill in the hero name, project details, and contact info.

### Phase 2: Repository Setup
1. Initialize a new local Git repository.
2. Create the GitHub repository (e.g., `manta-portfolio`).
3. Link and push the code to the repository.

### Phase 3: Deployment
1. Navigate to GitHub Repo Settings.
2. Enable GitHub Pages to deploy from the `main` branch.
3. Verify the live URL.

---

## Verification Plan
1. **Design Check:** Ensure IBM Plex Sans renders correctly and the dark theme contrast is WCAG-compliant.
2. **Functionality Check:** Ensure project cards are responsive on mobile and desktop.
3. **Deployment Check:** Confirm the site is live and all assets (CSS) are linked via relative paths.
