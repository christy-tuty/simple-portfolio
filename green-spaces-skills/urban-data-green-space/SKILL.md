# Urban Data Research for Green Space Development

This skill provides specialized workflows for using data to drive public green space initiatives.

## Core Workflows

### 1. Site Suitability Analysis
When asked to find the best location for a new park:
1. Identify high-density residential areas with low existing green cover.
2. Cross-reference with Land Surface Temperature (LST) data to find urban heat islands.
3. Search for vacant public land or underutilized grey spaces in those priority zones.

### 2. Accessibility & Equity Audit
When evaluating current green space distribution:
1. Apply the **3-30-300 rule** as a benchmark.
2. Calculate "walkability" buffers (300m) around existing parks.
3. Identify "green deserts"—neighborhoods where residents lack access to a 5-minute walk to a park.

### 3. Mechanism Framework Design
When designing a development strategy:
1. Use the **Green Space per Capita** metric (WHO standard: 9m²).
2. Propose **Pocket Parks** for high-density areas where large-scale land is unavailable.
3. Define KPIs for **Ecosystem Services** (cooling effect, stormwater reduction).
